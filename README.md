### Installation

Setup:

```sh
git clone rust-shopify-panel
cd rust-shopify-panel
npm install
```

- Please install and lauch node.js & mongodb before running anything. 

- Rename `config.example.json` to `config.json` when you're done.

After Setup:

```sh
node server.js
```
